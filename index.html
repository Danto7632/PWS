RAG 기반 AI와 함께하는 실전 고객 서비스 교육 플랫폼
신입 직원의 빠른 업무 적응을 돕는 인터랙티브 시뮬레이션

시연 영상 보기 GitHub 저장소

기획 배경 및 개요

문제 인식

  • 신입 직원의 고객 응대 업무 적응에 평균 3~6개월 소요
  • 실제 고객 대응 전 충분한 연습 기회 부족
  • 기존 교육 방식(매뉴얼, 선배 동행)의 한계
  • 업무 매뉴얼 기반 실전 시나리오 부재

해결 방안

  • 실제 업무 매뉴얼을 학습한 AI와 대화 시뮬레이션
  • RAG 기반 정확한 매뉴얼 참조 응답 생성
  • 고객/직원 역할 전환으로 다양한 관점 학습
  • 다양한 LLM 지원으로 유연한 환경 구성

주요 기능

프로젝트 관리

업무별 프로젝트 생성 및 매뉴얼 업로드, 지침 설정으로 체계적인 교육 환경 구성

문서 임베딩 (RAG)

PDF, Excel, Word, TXT 파일을 벡터 DB에 저장하여 AI가 매뉴얼 기반 응답 생성

역할 기반 시뮬레이션

고객 역할(AI 직원과 대화) 또는 직원 역할(AI 고객 응대)로 양방향 학습

Multi-LLM 지원

OpenAI, Gemini, Claude, Perplexity, Ollama 등 다양한 AI 모델 선택 가능

사용자 인증

JWT 기반 로그인/회원가입, 사용자별 프로젝트 및 설정 분리 관리

대화 기록 관리

모든 시뮬레이션 대화 저장, 복습 및 피드백 기능으로 학습 효과 극대화

기대 효과

업무 효율성 향상

  • ✓
    교육 기간 단축

    기존 3~6개월 → 1~2개월로 신입 직원 업무 적응 기간 50% 이상 단축

  • ✓
    24시간 학습 가능

    시간/장소 제약 없이 언제든 반복 연습 가능

  • ✓
    교육 담당자 부담 감소

    AI가 기본 교육을 담당하여 선배 직원의 업무 부담 경감
