# 🎯 실전형 업무 시뮬레이터 for 신입

신입 직원과 알바생을 위한 AI 기반 고객 응대 연습 도구입니다. 실제 업무 매뉴얼을 학습한 AI와 함께 다양한 상황을 시뮬레이션하며 실전 경험을 쌓을 수 있습니다.

## 🚀 주요 기능

### 📚 다양한 문서 지원
- **PDF**: 고객 응대 매뉴얼, 서비스 가이드
- **Excel**: FAQ, 상품 정보, 가격표  
- **TXT**: 간단한 업무 지침서
- **실시간 학습**: 업로드한 매뉴얼을 AI가 즉시 학습

### 🎭 양방향 역할 연습
- **👤 고객 역할**: AI 직원과 대화하며 서비스 흐름 체험
- **👔 직원 역할**: AI 고객의 다양한 문의에 실전 응대 연습
- **🔄 역할 전환**: 언제든 고객↔직원 입장 바꿔가며 이해도 증진

### 🎯 실전 중심 시뮬레이션  
- **매뉴얼 기반 시나리오**: 실제 업무 문서에서 자동 생성
- **다양한 고객 유형**: 일반/까다로운/급한 고객 등 맞춤 상황
- **실시간 피드백**: AI의 상세한 평가와 개선점 제시
- **학습 통계**: 연습 횟수, 평균 점수 등 성과 추적

## 🎪 시작하기

### 1단계: 환경 설정
```bash
# 의존성 설치
pip install -r requirements.txt

# Ollama 설치 및 모델 다운로드
ollama pull exaone3.5:2.4b
```

### 2단계: 애플리케이션 실행
```bash
streamlit run app.py
```

### 3단계: 시뮬레이션 시작
1. � 업무 매뉴얼 파일 업로드
2. 📖 "매뉴얼 학습 시작" 클릭  
3. 🎯 역할 선택 (고객 or 직원)
4. 💬 실전 대화 연습 시작!

- **Frontend**: Streamlit (웹 인터페이스)
- **AI Engine**: Ollama (로컬 LLM)
- **Vector DB**: ChromaDB (지식 베이스)
- **Embedding**: SentenceTransformers
- **파일 처리**: PyPDF2, pandas, openpyxl

### 파일 업로드 실패
- 지원 형식 확인: PDF, TXT, XLS, XLSX
- 파일 크기: 10MB 이하 권장
- 파일명: 한글 및 특수문자 주의

### 성능 최적화
- 매뉴얼 파일 크기 조정 (5MB 이하 권장)
- 불필요한 페이지 제거
- 핵심 내용 위주로 업로드
